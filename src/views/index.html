This is the default route. You get here if you use the browser to open a path that is not a valid GET request path.

See examples of valid requsts below:

curl -X PUT \ http://localhost:8210/hero -H 'content-type: application/json' \ -d '{ "name": "Batman", "strength": 8 }'

curl -X GET \ http://localhost:8210/hero/Batman

curl -X GET \ http://localhost:8210/hero/list

curl -X DELETE \ http://localhost:8210/hero/Super%20Woman
